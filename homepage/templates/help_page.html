How to Use Our Website

Display Items Search Items Pagination & Sorting View Details RTI Viewer Upload Items Usage & Sharing

Displaying Collections

Our database hosts visual data related to items research has been made on. You can browse these items from our main page which displays them in a user-friendly gallery format.

Searching for Items

You can search for specific items using the search form at the top of the homepage. You can look for an item using its title, the name of its author or the tags of the item.

Pagination and Sorting

To aid in navigation, our website implements pagination and sorting features. You can navigate through pages using the pagination controls at the bottom of the page, and you can sort items by various criteria using the sorting options provided (chronological and alphabetical).

Viewing Item Details

Click on any item in our collection to view more detailed information about it. The detail page will provide comprehensive information about the item. It also contains an image carousel at the top, allowing you to view the different still images associated with the item. Simply click one of the image thumbnails at the bottom to view the image in full size. Each image can have a short description that will be displayed alongside it.

Using the RTI Viewer

Our site features an RTI Viewer which allows you to interact with RTI files for an in-depth examination of certain collectibles. This tool helps you to explore every aspect of the item in high detail. You can access it by clicking the link at the top left of the image carousel.

Uploading new data

You can upload new items and related files to the database. In order to do so, just access the "upload form" page and enter the needed metadata alongside the files you upload. You can upload RTI data in the form of a RTI file(.ptm, .hsh). The RTI visualisation tool used is Openlime, which accepts .tzi, .dzi, and still image files as RTI planes.
